# Password Security Checker

A Python-based password security analysis tool that evaluates password strength using multiple security rules and provides detailed feedback to users.

## Features

* Password length validation
* Uppercase letter detection
* English letter verification
* Special character validation
* Weak/common password detection
* Username similarity checks
* Character substitution detection
* Birth year warning system
* Security score calculation
* Password strength classification

## Security Levels

* Very Strong
* Strong
* Medium
* Weak
* Very Weak

## Technologies Used

* Python
* Regular Expressions (re)
* Secure Password Input (getpass)

## Learning Outcomes

This project demonstrates:

* Python Functions
* Input Validation
* Regular Expressions
* String Manipulation
* Security Concepts
* Conditional Logic
* Error Handling
